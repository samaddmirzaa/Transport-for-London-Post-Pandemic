# London Public Transport Post-Pandemic Analysis

This project analyses how London Underground travel changed after the COVID-19 pandemic.

The goal of the project is to understand how ridership recovered over time, which stations and lines performed best or worst, and how passenger behaviour changed in the post-pandemic period. 
The analysis focuses on key themes such as station recovery, line performance, changing mode share, and the impact of hybrid working on weekday travel patterns.

Using TfL data, the project explores trends in Underground journeys across multiple years and compares recovery levels across stations and transport modes. 
It also highlights how some parts of the network recovered faster than others, showing that post-pandemic demand has not returned evenly across London.



## Key Analysis Areas
- London Underground journey trends over time
- Station-level recovery after the pandemic
- Best and worst performing stations
- Line performance comparison
- Changes in London transport mode share
- Evidence of changing commuter behaviour, including weaker Monday demand

## Project Outcome
This project shows that London Underground recovery after COVID was uneven. 
Some stations and travel patterns recovered strongly, while others remained below pre-pandemic levels.
The results suggest that hybrid working and shifting commuter habits have had a lasting impact on travel demand across the network.
